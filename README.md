# Obstacle-Avoidance-Robot
This project demonstrates an Obstacle Avoidance Robot built using purely analog electronics without a microcontroller. The robot uses IR sensors to detect obstacles and generates PWM signals using analog circuits such as a Schmitt trigger, integrator, and comparator to control the motors.
The system automatically changes direction when an obstacle is detected, allowing the robot to navigate safely.
