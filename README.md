# Obstacle-Avoidance-Robot
This project implements an automatic night lamp circuit using a Light Dependent Resistor (LDR). The lamp automatically turns ON during darkness and OFF during daylight based on the intensity of light falling on the LDR. The circuit uses basic electronic components such as a transistor, resistors, and an LDR sensor to control the lamp automatically.
